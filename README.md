## 核心技术亮点
1.  全场景缓存高可用方案
   - 基于Redis实现热点店铺、商品数据缓存，封装统一缓存客户端工具类
   - 集成布隆过滤器，解决缓存穿透问题，杜绝无效数据库请求
   - 基于互斥锁+逻辑过期方案，解决热点key缓存击穿问题
   - 随机过期时间策略，防止缓存雪崩，保障高并发下系统稳定性
2.  分布式锁秒杀系统
   - 集成Redisson分布式锁，实现可重入锁、看门狗自动续期，解决秒杀场景订单超卖问题
   - 优化秒杀流程，结合异步处理，将接口响应时间从300ms优化至50ms
3.  消息队列异步削峰
   - 集成RabbitMQ替换原有同步订单处理机制，实现秒杀订单异步创建、消息通知
   - 优化MQ配置与消息处理流程，修复消息丢失问题，保障消息可靠性
   - 削峰填谷，大幅降低峰值场景下的数据库压力，系统QPS提升400%
4.  业务扩展与工程化
   - 扩展实现博客发布、用户关注/粉丝列表、店铺地理位置查询等业务模块
   - 遵循Angular Git提交规范，分模块迭代开发，代码结构清晰，可维护性强
   - 基于Spring Boot3新特性适配，优化代码结构，提升项目性能
